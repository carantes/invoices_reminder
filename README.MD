# InvoicesReminder

Read customer data from a CSV file and sends out reminders of unsettled invoices based on the specified schedule. 

### Requirements

* NodeJS


### Installing

```
npm install
```

### Running

```
npm start
```

> http://localhost:3000

## Running the tests & code lint

```
npm run check 
```
